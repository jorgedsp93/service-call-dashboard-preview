# July Service Call Dashboard

Static dashboard for July service-call progress, trade mix, weekly pace, and July trade year-over-year comparisons.

## Run locally

```sh
python3 -m http.server 5173 --bind 127.0.0.1
```

Open `http://127.0.0.1:5173/`.

## Deploy

This is a static site. Use `index.html` as the entry point. No build command is required.
